### 專案說明
* 使用 Vuex 管理資料狀態
* 使用 Vue Router 管理路由
* 使用 Vue I18n 新增多語系內容
* 使用 JSON Web Token 實作驗證機制
* 使用 Axios 串接 API 資料
* 使用 VeeValidate 驗證表單內容
* 使用 Bootstrap 5 切版

### 主要功能
* 帳戶創建、登入、登出
* 快速訂房
* 選擇房型、預訂日期
* 即時顯示訂房價格總價
* 使用輪播效果提供各房型資訊
* 訂單篩選、編輯、刪除
* 支援 RWD 響應式網站 iPad Mini iPhone、12 Pro 等解析度
